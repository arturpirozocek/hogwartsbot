# Hogwarts School Web App

Одностраничное Telegram Web App для школьной RPG во вселенной Hogwarts.

## Как запустить локально

```bash
python3 -m http.server 4173
```

Откройте: http://localhost:4173

## Что реализовано

- Главная с 4 факультетами и очками.
- Разделы: Устав, Персонал, Еженедельный сплетник.
- Библиотека с 100+ книгами, плюс справочники по заклинаниям, зельям и существам.
- Профиль на основе Telegram ID и username.
- Ролевая модель прав: студент / преподаватель / деканы / администрация.
- Управление очками факультетов (по ролям).
- Управление расписанием по курсам 1–7 (по ролям).
- Управление квиддич-турниром (администрация): открыть/закрыть, пары, победитель, автоначисление очков.
- Подача материалов в сплетник студентами и публикация администрацией.

## Синхронизация с Telegram-ботом

В приложении предусмотрен слой API (`botApiBase`) для синхронизации данных.

Для рабочей синхронизации нужен backend (например, Node.js/Express), который:

1. Принимает запросы из Web App (`/house-points`, `/lessons-update`, `/gazette-publish`, `/quidditch-status` и т.д.).
2. Валидирует Telegram initData (подпись пользователя).
3. Сохраняет данные в БД.
4. Пишет в вашего бота `@school_hog_bot` через Bot API.
5. Возвращает обновлённые данные в Web App.

Без backend приложение работает в локальном режиме через `localStorage`.
