# Hogwarts Web App (пересобрано с нуля)

Готовое приложение в стиле Хогвартса: тёмно-золотой интерфейс, атмосферная библиотека и роли через Telegram Web App.

## Быстрый запуск

1. Скачай папку проекта.
2. Открой терминал в папке.
3. Запусти:

```bash
python3 -m http.server 4173
```

4. Открой в браузере:

http://127.0.0.1:4173

## Что уже встроено

- Главная «Замок» с 4 факультетами и песочными часами очков.
- Устав школы (аккордеон по пунктам).
- Персонал Хогвартса (администрация, деканы, преподаватели).
- Еженедельный сплетник (предложения от студентов + публикация по ролям).
- Уроки по курсам 1–7 (по умолчанию пустые; заполняются админами и преподавателями).
- Библиотека в «книжном» стиле:
  - разделы: заклинания, зелья, существа, волшебные предметы, учебные предметы, история, сказки/рассказы;
  - пергаментный ридер с главами;
  - изображения в книгах (особенно для зелий/существ/предметов);
  - расширенный каталог (каноничные тома + дополнительные учебные тома);
  - отдельные карточки-тома для списков существ и предметов с удалением повторов.
- Профиль через Telegram ID / username.
- Роли и права (student / teacher / admin).
- Управление факультетскими очками.

## По источникам контента

Книжные описания опираются на канон вселенной Harry Potter и структуру тематических разделов (магия, зелья, существа, предметы, история), совместимую с материалами Harry Potter Wiki/Fandom.

## Синхронизация с ботом

Приложение работает сразу в локальном режиме.
Чтобы данные были общими для всех пользователей, укажи backend API URL в профиле.
